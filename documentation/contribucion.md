# Guía de Contribución

## Requisitos
- Python 3.10+, Ultralytics, PyTorch compatible con CUDA (opcional).

## Estilo
- Mantener nombres claros y mensajes de commit descriptivos.
- No incluir secretos (API Keys) ni datos sensibles.

## Flujo sugerido
- Crear issue → fork → branch → pull request.
- Adjuntar evidencia (logs, ejemplos, capturas) cuando corresponda.
