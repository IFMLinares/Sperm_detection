# Roadmap Fase 2

## Objetivos
- Conteo y métricas por campo de visión.
- Clasificación morfológica y motilidad.
- Exportación a CSV/JSON y cuadro resumen por muestra.
- Interfaz mínima de QA.

## Tecnologías sugeridas
- Ultralytics para detección; clasificadores adicionales (CNN ligera) para morfología.
- Pandas para reporte; Streamlit/Gradio para UI simple.
